### **Example:**

> **Creating your own classes:**

```css
/* your input style.css */
@tailwind base;
@tailwind components;
@tailwind utilities;
@layer components {
  /* custom css class */
  .card-shadow {
    box-shadow: 0px 1rem 2rem -0.2rem hsl(224deg 23% 55% / 30%);
  }
}
```

> **Usage:**

```html
<!-- Inside html file -->
...

<div class="card-shadow">...</div>
```
